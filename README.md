# matasano-clj

A work in progress. Clojure solutions to the Matasano Crypto Challenge.

https://cryptopals.com/

Done to supplement learning more Clojure with some programming exercises.

## Usage
Requires Leiningen.
https://leiningen.org/

Confirm functionality with:
  `lein test`
